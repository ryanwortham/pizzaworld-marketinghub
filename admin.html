📊 Enable Remote Login Tracking (Google Sheets)

To track logins across all devices in a shared Google Sheet, follow these steps:

1. Go to Google Sheets → Create a new spreadsheet → Name it "Pizza World Login Log"

2. In the spreadsheet, go to Extensions → Apps Script

3. Paste this code:

function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([data.dateStr, data.timeStr, data.username, data.name, data.role, data.browser, new Date()]); return ContentService.createTextOutput('OK'); }

4. Click Deploy → New deployment → Web App → Set "Who has access" to "Anyone" → Deploy

5. Copy the Web App URL and paste it into the WEBHOOK_URL variable in login.html

All future logins will automatically appear in your Google Sheet in real time.
